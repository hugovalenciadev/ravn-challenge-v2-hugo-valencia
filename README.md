## Installation

```bash
$ yarn install
```

## Running the app

```bash
# set environment variables
# view .env.example

# run prisma migrations
$ yarn prisma migrate dev

# development
$ yarn start

# watch postman collection

```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
